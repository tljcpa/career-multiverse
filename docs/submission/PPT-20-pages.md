**对比表**：

|---|---|---|

| 反事实 | **真实重跑 + 统计聚合** | 关键反事实（项目 +15 / 简历 -20）真跑数十次校准，其余用单调插值；不对每个 mutation 都真跑，成本可控 |
| 存储 | **JSON + 文件锁 + 原子写** | 不上 SQLite / Postgres：约 300 公司 + 约 2000 persona 规模文件 MB 级，简单胜过复杂 |
| 数据 | **全代号 + sanitizer 双层** | 不做真公司数据：合规 0 风险优先 |

## 第 16 页 — 商业模式（评审"商业价值 20%" 拿分页）

**主图：一条价值链（横向箭头）**
学生免费用沙盘 → 数据让人才市场模拟更准 → 高校买"群体洞察"（主营收）→ 同一份数据再卖给企业 / 平台（第二曲线）→ 数据飞轮越滚越深

**三栏（主次分明，紫青粉）**：

**底部小字**：一份人才市场数据，对学校是洞察、对企业是品牌、对平台是匹配——边际变现成本极低。
