## 全局设计系统（先在 PowerPoint 幻灯片母版里设好，再做每页）

**画布**：16:9。背景深空渐变——左上 `#0A0E1A` → 右下 `#131829`；稀疏星点（低透明度小圆点，别多）。

**配色（4 色克制使用）**：
- 主青 `#22D3EE`（标题高亮、主数据）
- 紫 `#A855F7`（次级强调、架构节点）
- 粉 `#EC4899`（对比/警示数据）
- 金 `#FBBF24`（关键结论、钱相关）
- 文字：主 `#E5E9F0`，次 `#8B93A7`

**卡片（本版视觉核心，替代文本框平铺）**：
- 圆角 14px；填充 `rgba(255,255,255,0.04)`；1px 边框 `rgba(34,211,238,0.18)`；微投影（模糊 20、透明度 25%、下移 6）
- 内边距 24px；卡片之间间距 ≥24px
- PowerPoint 做法：插入「圆角矩形」→ 调圆角控点 → 设填充/线条/阴影，存为"默认形状"复用

**字体**：思源黑体 / 微软雅黑。标题 40-52pt Bold；副标题 22-24pt；正文 16-18pt；大数据 72-96pt Bold（配青/金）。

**铁律**：无 emoji、无任何非文本符号图标（用形状/线条替代）；每页正文 ≤40 字；每页 ≤3 个数字；不用 AI 生成配图。
