## 二、原创性声明

本人郑重声明，参赛作品《春招平行宇宙》系本人独立构思、设计与开发，为原创作品。具体说明如下：

1. **核心创意原创**：作品的产品形态（求职决策预演引擎：多 Agent 模拟 + 数十次真实平行春招统计推断 + 反事实分析）、产品结构与商业设计，均为本人原创，不抄袭、不剽窃任何第三方已有产品或方案。

2. **代码原创**：作品的后端（FastAPI 多 Agent 模拟引擎、聚合统计、反事实估计）、前端（Vue 3 + Three.js 3D 沙盘 + D3 可视化）及全部业务逻辑代码，均由本人编写，已开源于上述代码仓库，提交历史可追溯。

3. **AI 辅助声明（如实披露）**：本人在开发过程中使用了 AI 编程助手（Claude Code）作为协作工具，用于代码实现、调试与文档撰写。AI 仅作为工程实现的辅助手段，作品的核心创意、产品方向、架构决策与最终成果均由本人主导和负责。演示视频的旁白由 Azure Speech 文字转语音合成、画面由 Playwright 自动录屏，工具仅用于制作，不改变作品的原创属性。

4. **数据合规**：作品中的约 300 家公司、约 2000 名求职者画像均为**虚构合成数据**，公司均使用代号，不指向任何真实企业；不采集、不存储任何真实用户的个人隐私数据。详见随附《数据合规声明》。

5. **第三方组件**：作品依赖的开源框架与库（FastAPI、Vue、Three.js、D3.js 等）均遵循其各自的开源许可证使用，相关版权归原作者所有。
